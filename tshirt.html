Rotation 2: T-Shirt



Project Description:


My second project in this class was to design somthing to print on a t-shirt. I used Corel Draw to make my design. There is no significance to designing a bike. I found this project fun and easy to do. This was my first time using the Brother GT-3 machine. I like that we can design a custom shirt and have the ability to print it right away. Some limitations for the Brother GT-3 would be the size of the object and the surface. If the design is too big, then it won't fit on the shirt, and if the shrit is bumpy, then the print quality will be poor.
